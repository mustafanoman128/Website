Bangalore House Price
Prediction

This machine learning project predicts house prices in Bangalore using features like area (sqft), BHK, bathrooms, and location. Built with Python and Scikit-learn, it employs a Linear Regression model trained on historical data from the Bengaluru House dataset. The frontend is a Flask web app with HTML/CSS/JS for user input, and the model is serialized with Pickle for deployment.

0.86 R² Score

Key Techniques
  • Data cleaning & feature engineering on raw housing data
  • Dimensionality reduction of categorical location variables

  • Pickle serialization for production model serving

Healthcare Insurance
Cost Prediction

Using the insurance dataset, this project predicts individual medical costs using Random Forest regression. Features include age, sex, BMI, children, smoker status, and region. The Jupyter Notebook covers full data exploration (histograms, correlation matrices via Seaborn), encoding categorical variables, train-test split, model training, and evaluation using MAE and RMSE metrics.

~85% Accuracy
PCA Dimensionality Reduction
6 Input Features
Key Techniques
  • EDA — distributions of age, BMI, charges using Matplotlib & Seaborn
  • One-Hot Encoding for categorical variables (sex, region, smoker)
  • PCA for dimensionality reduction before modeling
  • Hyperparameter tuning via GridSearchCV
  • Model evaluation: MAE, RMSE, R² Score, feature importance plots
